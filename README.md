# TG WS Proxy — Releases

Канал распространения релизов Android-приложения **TG WS Proxy** — локального
прокси для Telegram с маскировкой через FakeTLS и обходом блокировок через
Cloudflare.

Сам репозиторий содержит только скомпилированные APK-файлы; исходный код,
issue tracker и история коммитов живут отдельно.

## Скачать

Последняя сборка: [Releases](../../releases)

Подойдёт устройство с Android 8.0 (API 26) или новее.

После первой установки приложение само начнёт проверять новые релизы и
покажет уведомление, когда выйдет свежая версия.

## Установка

1. Скачать APK со страницы [Releases](../../releases).
2. Открыть файл — система предложит включить установку из неизвестных
   источников для приложения, через которое скачали (Chrome, Telegram,
   менеджер файлов).
3. После установки можно отключить разрешение обратно — приложение
   попросит его снова, когда придёт время следующего обновления.

## Исходный код

- Android-приложение: репозиторий приватный, обращаться к
